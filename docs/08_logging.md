# 📘 ログ設計テンプレート

---

# 設計前提

| 項目     | 内容                       |
| ------ | ------------------------ |
| 対象システム | Web / API  |
| ログ方式   | 構造化ログ（JSON）必須 |
| 保持期間   | 30日  |
| 個人情報   | ログに含めない |

---

# ログ分類

| 種別                 | 目的        | 出力対象   |
| ------------------ | --------- | ------ |
| Application Log    | 動作確認・デバッグ | 開発・運用  |
| Access Log         | リクエスト追跡   | 運用     |

---

# ログレベル定義

| レベル   | 用途          |
| ----- | ----------- |
| DEBUG | 詳細情報（本番無効可） |
| INFO  | 正常動作        |
| WARN  | 想定内の異常      |
| ERROR | 処理失敗        |
| FATAL | サービス停止級     |
